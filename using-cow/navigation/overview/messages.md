# Messages

This page will display the messages for your COW instance.

## Table Details

The messages that appear in this table are related to subscription events, failed login attempts, etc. For sync related information, please see [logs.md](../logs.md "mention").

## Table Actions

<table><thead><tr><th>Action</th><th>Description</th><th data-type="checkbox">Bulk Action Available</th></tr></thead><tbody><tr><td>Mark as Read</td><td>Marks the selected message(s) as read and stops displaying them on the <a data-mention href="./">.</a>page.</td><td>true</td></tr><tr><td>Remove Message</td><td>Removes the selected message(s) from the database.</td><td>true</td></tr></tbody></table>
